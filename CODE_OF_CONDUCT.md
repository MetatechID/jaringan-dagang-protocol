# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/),
version 2.1.

## In short

- Be respectful. Disagree with ideas, not people.
- Assume good faith. Most miscommunications are not malicious.
- No harassment, no doxxing, no slurs.
- Keep technical discussions technical.

## Enforcement

Report violations to security@metatech.id. Maintainers will
investigate and act — actions include warnings, temporary bans, or
permanent removal from the project. Reporters' identities stay
confidential.

The full text of the Contributor Covenant is at
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.
